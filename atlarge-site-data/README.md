# atlarge Site Data

This repository holds the data (on members, publications, and other topics) to be used by the [atlarge site build system](https://atlarge.ewi.tudelft.nl/gitlab/gandreadis/atlarge-site).


## Contributing

Please refer to the README of [the build system repository](https://atlarge.ewi.tudelft.nl/gitlab/gandreadis/atlarge-site) for instructions and guidelines on contributions in this data repository.

To propose a change, please fork this repo and submit a merge request. Once merged, the online version of this site will be re-compiled to reflect the change in data.
